Preguntas sobre el lenguaje

¿Qué es HTML?
  • Es el "HyperText Markup Language", un lenguaje de marcado para estructurar contenido web y no un lenguaje de programación.
¿Por qué se considera el lenguaje de la web?
  • Porque define la estructura de las páginas web y es interpretado por todos los navegadores.
¿Qué son las etiquetas?
  • Son el código adicional que se usa para dar formato y estructura al texto, como por ejemplo para un título o para un párrafo.
